What is Recursion?

Recursion is a programming technique where a function calls itself to solve a problem. It's particularly useful for tasks that can be broken down into similar subtasks.

The key components of recursion are:

  • Base case - the condition that stops the recursion
  • Recursive case - where the function calls itself
  • Progress toward the base case - each recursive call should move closer to the base case
